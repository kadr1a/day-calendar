# Календарь дня

Простое React-приложение для отображения текущей даты и управления заметками на день. Позволяет добавлять, просматривать и удалять ежедневные заметки
## Функционал

- Отображение даты
- Добавление, просмотр, удаление заметок
- Вывод времени создания заметки

## Технологии

- React
- CSS
- JavaScript
- HTML
